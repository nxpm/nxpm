title: Using nxpm cli
sidebar_label: Usage
---

Once installed you can run `yarn nxpm`

```shell script
$ yarn nxpm
nxpm cli

VERSION
nxpm/1.0.0 darwin-x64 node-v12.16.2

USAGE
$ nxpm [COMMAND]

COMMANDS
help display help for nxpm
release Release publishable packages in an Nx Workspace
```
